Popcorn (beta)

Popcorn is an implementation of a game known as Powerpoint Karaoke, or battledecks, in which the participant receives a set of random slides and must make a presentation. Here, we replace the slides with a set of five randomized sources (currently, in this beta, only Tanach and Talmud).
